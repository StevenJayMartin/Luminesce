# 🤝 Contributing to Lumin

Thank you for your interest in contributing to Lumin!  
Lumin is a local-first, hackable AI assistant with a glowing ASCII light-bulb face — and a bright future.

We welcome contributions of all kinds:
- Code
- Documentation
- UI/UX ideas
- Web interfaces
- Hardware integrations
- Bug reports
- Feature proposals

---

## 🧱 Project Philosophy

Lumin is:
- **Local-first** — no cloud dependencies  
- **Fun** — personality matters  
- **Hackable** — simple, readable code  
- **Modular** — core logic separated from interfaces  
- **Extensible** — plugins, APIs, and UI layers  

If your contribution makes Lumin more fun, more local, or more hackable — it’s welcome.

---

## 🛠 Development Setup

1. Clone the repo  
2. Install dependencies  
3. Create a `config.json`  
4. Run in TUI or API mode
