# 🌱 Good First Issues

These tasks are perfect for new contributors who want to help Lumin grow.

---

## 🟢 UI / TUI
- Add blinking animation to Lumin’s eyes
- Add color support to the ASCII bulb
- Add a “charging up” startup animation
- Add a volume meter under the bulb

---

## 🟣 API / Backend
- Implement FastAPI /chat endpoint
- Implement Flask web backend
- Add streaming responses to API mode
- Add CORS support for web UI

---

## 🔵 Core Assistant
- Add silence_threshold to config.json
- Add silence_duration to config.json
- Add auto-calibration for mic noise floor
- Add wake-word detection (Porcupine or VAD)

---

## 🟡 Web UI
- Build a simple HTML/JS frontend
- Add a glowing bulb animation in CSS
- Add push-to-talk button
- Add streaming text output

---

## 🟠 Documentation
- Improve README screenshots
- Add architecture diagram
- Add developer setup guide
- Add API documentation

---

If you want to help but aren’t sure where to start, open an issue — we’re happy to guide you.
