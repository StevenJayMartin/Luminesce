# 🗺️ Lumin Roadmap

This roadmap outlines the future of Lumin — a local-first AI assistant with multiple interfaces and a glowing personality.

---

## 🚀 Phase 1 — Core Assistant (Now)
- Stable STT → LLM → TTS pipeline
- Configurable via config.json
- TUI light-bulb interface
- Logging to lumin.log
- Argparse launcher

---

## 🌐 Phase 2 — API Mode
- FastAPI JSON API
- Flask web backend
- Web UI (HTML/JS)
- Cross-platform support (Windows, Linux, macOS)

---

## 🎨 Phase 3 — UI Enhancements
- Colorized bulb
- Blinking animations
- Volume meter
- Startup animation
- Personality packs

---

## 🔌 Phase 4 — Plugin System
- Skills directory
- Python plugin API
- Hot-reloadable modules
- Community skill marketplace

---

## 🏠 Phase 5 — Home Assistant Mode
- Wake-word detection
- VAD (voice activity detection)
- MQTT integration
- Smart home control
- Raspberry Pi deployment

---

## 🌍 Phase 6 — Ecosystem
- Docker images
- Systemd service
- Windows service mode
- Mobile web UI
- Hardware LED bulb integration

---

## ⭐ Long-Term Vision
Lumin becomes a fully local, modular, multi-interface AI assistant that can live:
- In your terminal  
- On your server  
- In your browser  
- In your home  
- On your devices  

A tiny glowing assistant with a bright future.
